Blackjack Game is a complete mobile game project.

If you want bo build a Blackjack mobile or web game, or you want to learn how to make 
a poker game, this is what you need.

Blackjack Game provides a complete game project example, 
including complete game features, 
as well as all the arts and audio resources.

Features:

* Optimized for Mobile devices
* Complate game logic
* Score Saving & Loading
* Sound Play & Mute control
* UI Animations
* Poker deal animation




------------------------------------------------------
INSTALLATION:
------------------------------------------------------
To install Blackjack Game simply import all files from the unitypackage.

------------------------------------------------------
REQUIREMENT:
------------------------------------------------------
This project use LeanTween for UI animations.
Please import LeanTween before run this project.

LeanTween: http://u3d.as/31i

------------------------------------------------------
HOW TO  RUN:
------------------------------------------------------
Open 'Assets\Blackjack Game\Scenes\MainScene.unity', click "Play" in unity editor.

------------------------------------------------------
DOCUMENTATION:
------------------------------------------------------
Assets\Blackjack Game\Documentation\Documentation.zip
Unzip and open index.html
